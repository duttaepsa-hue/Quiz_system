# Quiz_system
Objective: Developed an interactive quiz game using object-oriented programming, where users answer multiple-choice questions, and their scores are saved to a file. The game should handle invalid inputs gracefully to ensure a smooth user experience.

Skills & Tools: Python , Exception Handling, Error Management, OOP concepts

Resolution: Design a Quiz class to manage questions and scores. Implement a game loop to present questions and validate user input. Use exception handling to handle invalid inputs. Save user scores to a file. Provide feedback and scoring to the user.
